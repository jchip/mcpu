# Tacks

Lightweight issue tracker MCP server.

## Prerequisites

Install [bun](https://bun.sh):

```bash
curl -fsSL https://bun.sh/install | bash
```

## [MCPU](https://www.npmjs.com/package/@mcpu/cli)

```bash
mcpu add tacks --stdio "bunx @mcpu/tacks"
```

## Claude CLI

```bash
claude mcp add tacks -- bunx @mcpu/tacks
```
